# 🏘️ Barangay Census Online Survey System

A simple web-based Barangay Census Online Survey system built using **HTML, CSS, and JavaScript**.  
It provides an easy and organized way for residents to access information and participate in an online census survey.

---

## 📌 Features

- 🧭 Navigation menu (About, Help, Contact)
- 📝 Online survey access via “Start Survey” button
- ℹ️ About section explaining the system
- 🆘 Help section for user guidance
- 📞 Contact information for support
- 💻 Simple and user-friendly interface

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript

---

## 🚀 How to Use

1. Open the project folder
2. Run `index.html` in a browser
3. Navigate through the menu:
   - About
   - Help
   - Contact
4. Click **Start Survey** to proceed to the survey form

---

## 🎯 Purpose

This project was created to modernize barangay census data collection by providing a digital and accessible online survey system for residents.
